Overview

Smart Choice is a web application, which can help to choose the best option based on our own weighted rules.
This application is based on the framework called "Weighted Decision Making Matrix".
More information about this framework can be found in weighted decision matrix prioritization article.

Step by step user guide
1. Create the rule

The first step in the journey is to create a rule, which outlines the criteria to be used
for comparing options and on their weights / relative importance.

Navigate to Rules page and click "Create" button to start creating the new rule:

Add the criteria which we want to use and assign weights based on their relative importance:

Save the rule.

2. Create the comparison

Now that our rule is ready - navigate to Comparisons page and click "Create" button:

Give the comparison a meaningful name and select the rule we just created:

Fill the options and rate them based on the criteria from our rule:

Save the comparison.

3. See the comparison details

From Comparisons page click on the newly created comparison to see the details:
